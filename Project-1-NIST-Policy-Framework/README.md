# Project 1 — NIST Policy Framework

This directory contains materials for Project 1: implementing a NIST-based policy framework.

Contents:
- README.md (this file)
- policies/ — place policy documents (e.g., Information Security Policy, Access Control Policy)
- templates/ — templates for policy documents and assessments
- evidence/ — implementation evidence, such as screenshots, logs, or audit outputs

Purpose:
- Provide a centralized location for NIST Cybersecurity Framework (or relevant NIST guidance) policy development and artifacts for this project.

How to use:
1. Add policy documents to the `policies/` folder.
2. Use `templates/` for standardized policy formats.
3. Store implementation evidence in `evidence/`.
